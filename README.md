# ClinicManagementSystem
Java-based medical clinic management system using 2-3 Tree data structures for patient and doctor allocation.

## 📌 Overview
This project was developed as part of a **Data Structures and Algorithms** course.  
It implements an efficient system for managing a medical clinic, including doctors, patients, and waiting lists.

The goal is to handle dynamic operations while maintaining optimal performance using appropriate data structures.

---

## ⚙️ Features
- Add and remove doctors dynamically
- Register patients and assign them to doctors
- Manage waiting queues for each doctor
- Efficient handling of large-scale data (many doctors and patients)
- System initialization and reset functionality

---

## 🧠 Concepts & Techniques
- Object-Oriented Programming (OOP) in Java
- 2-3 Tree data structure for efficient allocation
- Time complexity optimization
- System design and modular programming
